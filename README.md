# Flight-Scanner
Search Flights powered by SkyScanner API

## Installation
1. Clone the repository
2. Install all the dependencies (`npm install`)
3. Run it locallly (`npm run dev`)
4. Navigate to [http://localhost:3000](http://localhost:3000)
